# Partage_de_fichiers

### 1. Création du partage pour le dossier docs
- Clique droit dans shares puis new share
- selectionne SMB Share Quick puis clique sur next
- choisis la partition souhaité pour le nouveau partage puis clique sur next
- Donne le nom a ton partage puis clique 2 fois sur next
- dans les permissions ajoute une permission pour tout le monde en ecriture/lecture puis next
- et clique sur create

### 1.2. Création des permissions pour les sous dossiers RH Comptabilité et Direction
- creer un nouveau partage
- dans share location choisi custom path puis choisi le dossier docs
- donne lui le nom souhaite exemple RH
- dans les permission créer une permissions d'écriture et de lecture pour le groupe RH créer au préalable
- next et create
Renouvel l'operation pour chaque dossier.
